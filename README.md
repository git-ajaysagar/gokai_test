# Raj Shukla — Simple Personal Site

This small static site shows name, designation, service and contact links.

How to customize
- Open [`index.html`](index.html) and replace the WhatsApp phone number in the `wa.me` link (format: country code + number, e.g. `91XXXXXXXXXX`).
- Replace the email in the mailto link.

Run locally
- Open `index.html` in your browser (double-click or drag into the browser).

Deploy
- Push to GitHub and enable GitHub Pages, or host the files on any static host (Netlify, Vercel, etc.).

Contact details
- WhatsApp link currently uses a placeholder number — replace it before sharing.
